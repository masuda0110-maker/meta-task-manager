# Meta-Task Manager

**Todoist互換の操作性 × AIメタ認知で自己省察を支援する次世代タスク管理アプリ**

---

## 📋 プロジェクト概要

- **アプリ名**: Meta-Task Manager
- **目的**: 日々のタスク管理を高速に行いながら、AIコーチとの対話で思考の質を高める
- **コンセプト**: タスク完了 → メタ認知振り返り → 継続的成長のサイクルを実現

---

## ✅ 実装済み機能

### 1. タスク管理（Todoist互換）
- 自然言語クイック追加（Ctrl+K）例:「明日10時に企画書作成 P1 #仕事 60分」
- 4段階優先度（P1〜P4）カラーコーディング
- プロジェクト別タスク管理（カラーラベル付き）
- 今日・近日予定（14日先まで）ビュー
- 繰り返し設定（毎日・平日毎日・毎週・毎月）
- サブタスク（チェックリスト）
- タスク詳細モーダル（編集・削除・サブタスク管理）
- 優先度フィルター（今日ビュー・プロジェクトビューに対応）
- 連続完了日数（ストリーク）表示

### 2. プロジェクト管理 + WBS（Work Breakdown Structure）
- プロジェクト作成・編集・削除（8色カラー）
- プロジェクトビューでタスク一覧タブとWBSタブを切替
- **WBS ツリービュー**:
  - フェーズ（ルートノード）追加
  - 階層構造（無制限ネスト）
  - 子タスク追加・編集・削除
  - ドラッグ＆ドロップで順序変更
  - ノード展開/折りたたみ
  - ステータス管理（未着手・進行中・完了・ブロック）
  - 優先度設定（P1〜P4）
  - 進捗率バー（0〜100%）
  - 担当者・開始日・終了日・見積/実績時間
  - タスクリストとの連携（WBSノード↔タスク紐付け）
  - 深さによる色分け（5色）
- **WBS ガントチャートビュー**:
  - 月・日ヘッダー表示
  - 今日ラインの表示
  - 土日ハイライト
  - バー上のクリックで編集モーダル起動
  - 進捗率のオーバーレイ表示
- **WBS 統計バー**: 総タスク数・完了・進行中・ブロック件数・進捗率・見積/実績時間

### 3. メタ認知エンジン

#### 手動データ追加（「+ 記録する」ボタン）
- メタ認知ダッシュボード上部の「**＋ 記録する**」ボタンから任意のタイミングで記録可能
- **記録タイプ選択**: タスク完了 / 朝セッション / 夜の振り返り / フリー の4種類
- **タスク名入力**（任意）
- **集中度** — 5段階スター評価
- **時間見積もり精度** — 早く終わった / ほぼ予定通り / 少し超えた / 大幅に超えた
- **阻害要因** — 複数選択可（集中できなかった・割り込み・情報不足・疲労・技術的問題）
- **学んだこと・気づき** — 自由記述
- **次回への意図** — 次回改善したいことを記録
- **エネルギーレベル** — 高い / 普通 / 低い
- 保存後、ダッシュボードに即反映

#### メタ認知ダッシュボード（サイドナビ「メタ認知ダッシュボード」）
- **スコアサマリー行**: メタ認知スコア(0〜100)/平均集中度/時間見積もり精度/振り返りセッション数/連続日数
- **集中度トレンドチャート**: タスク完了時の★評価を時系列で可視化（Chart.js ラインチャート）
- **メタ認知レーダーチャート**: 自己認識・計画性・適応力・内省力・集中管理・時間感覚の6軸評価
- **時間見積もり精度チャート**: 「早く終わった/予定通り/超えた」の割合をドーナツチャートで表示
- **思考癖パターン分析**: 振り返りテキストから6種の思考癖を検出 + 改善アドバイス
- **蓄積された気づき**: コーチとの会話やタスク完了後の気づきを一覧表示
- **振り返りログ一覧**: 種別フィルター・集中度・時間精度タグ付き・クリックで詳細表示・削除可能
- **期間切替**: 今週 / 今月 / 全期間
- **未読バッジ**: 新しい振り返りログがある時に表示
- **レポートエクスポート**: Markdownファイルとしてダウンロード可能
- 初回起動時にサンプル振り返りデータが自動追加（デモ用）

### 4. 朝夜ブリーフィング
- 17:00 を境に朝・夜パネル自動切替
- **朝ブリーフィング**: P1タスク一覧・課題分析（期限切れ・P1集中・未プロジェクト率）・NEXTタスク推奨（最大4件）
- **夜の振り返り**: 今日の完了タスク・持ち越しタスク・明日へのNEXT提案

### 4. メタ認知ダッシュボード（NEW）
- **メタ認知スコア**（0〜100点）: セッション数・集中度・時間精度・気づき記録・種別多様性から総合算出
- **集中度トレンドグラフ**: タスク完了時の★評価を時系列で可視化（Chart.js ラインチャート）
- **メタ認知レーダーチャート**: 自己認識・計画性・適応力・内省力・集中管理・時間感覚の6軸評価
- **時間見積もり精度チャート**: 「早く終わった/予定通り/超えた」の割合をドーナツチャートで表示
- **思考癖パターン分析**: 振り返りテキストからAIが6種の思考癖を検出（完璧主義・先延ばし・考えすぎ等）+ 改善アドバイス
- **蓄積された気づき**: コーチとの会話やタスク完了後の気づきを一覧表示
- **振り返りログ一覧**: 種別フィルター（タスク完了/朝/夜/フリー）・集中度・時間精度タグ付き
- 期間切替：今週 / 今月 / 全期間
- 未読バッジ（新しい振り返りログがある時に表示）

### 5. AIメンタリングコーチ（チャットUI）
- **5モード**: フリー相談・朝セッション・夜の振り返り・タスク完了後・**メタ認知セッション**
- **🤖 Gemini AI対応**: 設定画面でAPIキーを入力するとGemini 2.0 Flashによる本物のAI会話が可能
  - APIキーなし → ルールベースで自動フォールバック（いつでも使える）
  - APIキーあり → Gemini APIで文脈を理解した自然な返答
  - 対応モデル: Gemini 2.0 Flash（推奨・無料枠あり）/ 1.5 Flash / 1.5 Pro
- コンテキスト対応（現在のタスク状況・過去の振り返りをAIに自動伝達）
- クイック返信チップ（状況依存のサジェスト）
- 蓄積された洞察（週次・月次表示）
- 7日間成長トレンドグラフ（Chart.js）
- セッション履歴
- タスク完了後の自動メタ認知チェックモーダル
- ブリーフィングボタンからAIコーチへのシームレス遷移

### 5. 連携機能
- **Googleカレンダー**: ワンクリックでタスクをイベント追加（URLスキーム）
- **Outlook**: ICSファイル生成・ダウンロード
- **Slack**: 個別タスク通知・自動デイリーブリーフィング（朝/夜、設定時刻にWebhook送信）

### 6. データ永続化
- Table API（RESTful）への読み書き
- LocalStorageフォールバック（オフライン対応）

## 🔧 設定方法

1. サイドバー下部の歯車アイコン（設定）をクリック
2. **Slack Webhook URL** を入力して朝/夜の通知時刻を設定
3. **ユーザー名** を入力するとブリーフィングの挨拶に反映

---

## 📝 WBSの使い方

1. サイドバーのプロジェクトをクリック
2. 「**WBS**」タブをクリック
3. 「**フェーズ追加**」ボタンでルートフェーズを作成
4. 各行の「＋」ボタンで子タスクを追加
5. 行をドラッグして順序変更
6. 「ガント」ボタンでガントチャートビューに切替（開始日・終了日が必要）
